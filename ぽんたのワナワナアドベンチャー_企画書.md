## 1. ゲームコンセプト

### タイトル
**ぽんたのワナワナアドベンチャー**

### 主人公
**ぽんた**

### キャッチコピー
**「常識を捨てろ。このゲームに正解はない。」**

### コンセプト

| 項目 | 内容 |
|---|---|
| ジャンル | 理不尽系横スクロールアクション |
| ステージ数 | 全5〜6ステージ（まず1ステージから開発） |
| プレイ時間 | 1ステージあたり約5〜15分（死亡回数による） |
| プレイスタイル | 何度も死んで学ぶ・常識を疑う |

### 3つのコアバリュー

```
① 理不尽さ
　　見た目と効果が真逆のアイテム・罠・ギミックが
　　プレイヤーの常識を次々と裏切る

② 違った思考でのゴールの目指し方
　　一見正解に見えるルートが罠で
　　一見おかしいルートが正解という
　　逆転の発想が求められる

③ 発見と達成感
　　裏ルートの発見・ボス撃破・ゴール到達時の
　　達成感が次のプレイへの意欲を生む
```

## 4. ゲームの特徴・差別化ポイント

### 4-1. 理不尽ギミック（しょぼんのアクション要素）

見た目と効果が真逆のギミックでプレイヤーの常識を裏切ります。

| ギミック | 見た目 | 実際の効果 |
|---|---|---|
| キノコ | パワーアップアイテム（マリオの常識） | 触れると死亡 |
| 土管 | ワープポイント（マリオの常識） | 高く飛ばされてゲームオーバー |
| ブロック | 足場 | 乗ると崩れて落下死 |
| コイン | 安全なアイテム | 触れると罠が発動 |
| ゴールポール | ゴール | 近づくと爆発（フェイクゴール） |

### 4-2. 裏ルート（本作独自要素）

一見通れなさそうな場所に正規ルートとは別の裏ルートを設置します。

| 裏ルートの種類 | 内容 |
|---|---|
| 隠し壁 | 見た目は壁だが実は通り抜けられる |
| 逆走ルート | 左に戻ると実は近道がある |
| 落下ルート | 穴に落ちると実は下に安全な足場がある |
| 時間待ちルート | その場で待つと障害物が動いて道が開く |

### 4-3. ボス戦（本作独自要素）

各ステージのボスも理不尽な攻撃パターンを持ちます。

| ボスの特徴 | 内容 |
|---|---|
| 見た目が可愛い | 可愛い見た目なのに強力な攻撃をしてくる |
| 弱点が理不尽 | 通常攻撃では倒せず特定の行動で倒せる |
| フェイク死亡 | 一度倒れたように見えてから復活する |

### 4-4. パワーアップ要素（本作独自要素）

本作のパワーアップは**本当に効果があるものと嘘のものが混在**します。

| パワーアップ | 見た目 | 実際の効果 |
|---|---|---|
| 赤いアイテム | 危険そうな見た目 | 実は無敵になる |
| 星 | 無敵アイテム（マリオの常識） | 実は動きが遅くなる |
| 緑のアイテム | 安全そうな見た目 | 実はジャンプ力が下がる |
| ハテナブロック | アイテムが出る（マリオの常識） | 実は爆発する |

## 5. ゲームシステム

### 5-1. 基本操作

| 操作 | アクション |
|---|---|
| 左右キー / ADキー | 左右移動 |
| スペース / 上キー | ジャンプ |
| 下キー / Sキー | しゃがむ |
| しゃがみ + ジャンプ | 低い障害物をくぐる |

### 5-2. ゲームオーバー条件

| 条件 | 内容 |
|---|---|
| 敵に触れる | 即死 |
| 穴に落ちる | 即死 |
| 罠に触れる | 即死 |
| 画面外に飛ばされる | 即死 |

### 5-3. リトライシステム

| 項目 | 内容 |
|---|---|
| リトライ方法 | 即座にリトライ可能（ストレスを最小化） |
| チェックポイント | ステージ中盤に1箇所設置 |
| 死亡回数表示 | 画面上に死亡回数を表示（やりこみ要素） |

## 6. ステージ構成

### 6-1. ステージ一覧

| ステージ | テーマ | 特徴 |
|---|---|---|
| Stage 1 | チュートリアル・草原 | 基本操作を学びながら最初の理不尽に気づく |
| Stage 2 | 地下・洞窟 | 暗い場所で見えない罠が増える |
| Stage 3 | 空中・雲の上 | 足場が少なく落下死が増える |
| Stage 4 | お城 | 罠の密度が上がりボスが登場 |
| Stage 5 | 最終ステージ | すべての理不尽要素が組み合わさる |
| Stage 6（隠し） | 裏ステージ | 全ステージクリア後に解放される超難関ステージ |

### 6-2. Stage 1の詳細構成

```
【スタート地点】
　↓
【チュートリアルゾーン】
　・普通のジャンプ・移動を学ぶ
　・一見普通のステージに見える
　↓
【最初の罠ゾーン】
　・崩れるブロックが登場
　・キノコが出て触れると死亡（最初の理不尽）
　↓
【裏ルート分岐点】
　・正規ルート：難しい罠が続く
　・裏ルート：隠し壁を通ると簡単に進める
　↓
【チェックポイント】
　↓
【ボス前エリア】
　・土管が登場（入ると飛ばされてゲームオーバー）
　・フェイクゴールが登場
　↓
【ボス戦】
　↓
【本物のゴールポール】
```

## 10. UI・演出方針

### 10-1. UI方針

| UI要素 | 内容 |
|---|---|
| 死亡回数表示 | 画面左上にリアルタイム表示 |
| ステージ表示 | ステージ開始時にアニメーションで表示 |
| チェックポイント演出 | チェックポイント通過時にエフェクト表示 |
| ゲームオーバー画面 | コミカルな演出でリトライ・タイトルへ戻るボタンを表示 |
| ゴール演出 | ゴールポール到達時に派手な演出を表示 |

### 10-2. 演出方針

- 死亡時はコミカルな演出でストレスを軽減する
- ゴール到達時は達成感のある派手な演出を表示する
- 裏ルート発見時は特別なエフェクトを表示する
- すべてのボタン・アクションにSEを設定する

### 10-3. こだわりポイント

- **死亡アニメーション**：しょぼんのアクションのように吹っ飛んで死ぬコミカルな演出
- **罠発動時のエフェクト**：DOTweenを使った派手なエフェクト
- **ボス戦演出**：ボス登場時・撃破時に演出を追加
- **裏ルート演出**：隠しルート発見時に特別なエフェクトを表示
