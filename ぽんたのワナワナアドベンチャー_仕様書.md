## 1. システム仕様

### 1-1. 基本システム

| 項目 | 内容 |
|---|---|
| 画面解像度 | 1280 × 720 |
| フレームレート | 60fps |
| 重力スケール | Physics2D.gravity (-9.81) |

| 死亡後待機時間 | 0.6秒後に画面暗転開始 |

## 3. カメラ仕様

### 3-1. Cinemachine設定

| 項目 | 値 |
|---|---|
| カメラコンポーネント | CinemachineCamera |
| Position Control | Position Composer |
| Dead Zone Size X | 0.3 |
| Dead Zone Size Y | 0.9 |
| Damping X | 0.5 |
| Damping Y | 0.9 |

### 3-2. チェックポイント復活時のカメラリセット

1. vcam.Follow = null
2. カメラをtargetPos + cameraVerticalOffset(1.5)に強制移動
3. 1フレーム待機
4. vcam.Follow = playerTransform 再セット
5. 2フレーム待機
6. ForceCameraPosition を2回実行
7. フェードイン開始

| 踏みつけ速度 | 12.0 | 14.0 | 16.0 |
| トゲ発動数 | 6個 | 7個 | 8個 |

### 8-2. 状態遷移

```
Waiting（ぽんたが activateRangeX 以内）
  ↓ ボス戦開始・ボスBGM再生
Rising（上昇アニメ3枚・startPosition.y + maxFlyHeight まで上昇）
  ↓ ReturnToCenterX でfloatCenterPosへ戻る
Floating（左右移動・浮遊アニメ3枚ループ）
  ↓ spikeAttackInterval 経過
SpikeAttack（攻撃アニメ4枚→ランダム8ブロックトゲ発動）
  ↓ ReturnToCenterX で戻ってFloatingへ
Floating
  ↓ battleTimer >= nextStompTime
StompAttack（踏みつけ攻撃）
  ↓ 着地→Stunned（3秒間）
  ↓ 踏まれなかった場合→RiseSequence→Floating
  ↓ 踏まれた場合→TakeDamage→Hit（混乱アニメ）→RiseSequence→Floating
```
